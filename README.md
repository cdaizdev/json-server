# JSON Server API (Mock)
Fast API REST built with NodeJS. Use it to test your front-end, prototipes and learning. 

No website, no extras, just your own API

## Requirements
* **Node.js** (^v14)
* **npm** 

## Install
1. Replace db.json with your own db.json
2. Run the code!
```bash
node index.js
```
